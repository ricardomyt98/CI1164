◆ media()

double media ( double *  valores,
unsigned long  n 
)

Função para calcular a média de um vetor de números reais.

Possíveis erros de precisão: Devido a possivelmente haver uma grande variação entre os valores do vetor, a soma destes pode gerar propagação de erro a cada iteração. Uma maneira de minimizar este erro seria para cada valor do vetor, ir dividindo este por "n" ("padronizando" os valores para a soma), porém isto geraria o dobro de operações, podendo causar grande lentidão.

Possível erro de divisão: A soma dos elementos do vetor pode resultar em um valor muito grande e "n" ser muito pequeno, causando erro na divisão e vice-versa.

Parameters
valoresVetor com os números reais.
nQuantidade de elementos do vetor.
Returns
Resultado da média aritmetica.
